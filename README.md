BME280
============

Python library to use BME280 on Beaglebone Black

| Board pin name | Board pin | Beaglebone Black pin name |
|----------------|-----------| --------------------------|
| VIN            | 1         | P9\_05, VDD\_5v           |
| 3Vo            | 2         | Not Used                  |
| GND            | 3         | P9\_01, Ground            |
| SCK            | 4         | P9\_19, I2C2\_CL          |
| SDO            | 5         | Not Used                  |
| SDI            | 6         | P9\_20, I2C_SDA           |
| CS             | 7         | Not Used                  |
